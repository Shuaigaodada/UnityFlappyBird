# Flappy Bird Clone

这是经典游戏Flappy Bird的一个克隆版本，使用Unity开发。

## 游戏介绍

在这个游戏中，玩家需要控制一只小鸟，避免撞到上下移动的管道。玩家通过点击或敲击屏幕使小鸟向上飞行，不操作时小鸟会下落。游戏的目标是尽可能长时间地生存下去，同时穿越尽可能多的管道。

## 特点

- 简单易上手的控制方式
- 经典的游戏玩法
- 逐渐增加的难度

## 如何玩

1. 下载游戏
2. 解压缩文件
3. 运行`FlappyBird.exe`
4. 点击或敲击屏幕控制小鸟飞行
5. 尽量获得更高的分数！

## 构建

该游戏使用 Unity 2022.3.5f1 开发。如果你想自行构建游戏，可以克隆仓库并在Unity编辑器中打开。
